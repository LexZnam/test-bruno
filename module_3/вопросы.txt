1.1 Материнская плата - плата которая объединяет все другие компоненты компьютера и обеспечивает их взаимодействие.
1.2 Оперативная память  - это тип памяти, в котором во время работы компьютера хранится выполняемый машинный код и данные, которые в этот момент обрабатывает процессор.
1.3 Долгосрочная память - это память, предназначенная для долговременного хранения данных и программ.
2.1 Стек - это определенная область памяти, где при вызове функции данные появляются, а при ее завершении - удаляются. Память стека ограничена.
2.2 Куча - это область памяти в компьютере, которая используется для динамического выделения памяти.
В куче переменные необходимо создавать и удалять явным образом.
3 Типы данных в java в основном:
примитивные - целочисленные, символьные, с плавающей точкой и логические.
ссылочные типы - объекты и массивы.
4.1 Примитивные данные используются для хранения простой информации, например чисел, символов или логических значений.
4.2 Ссылочные не содержат значения, а ссылаются на место, где они расположены.
5 Переменная в Java - это контейнер,  в котором хранится значение данных для дальнейшего использования в программе.
6. Логические операторы ! , & и | ?

& логическое «и»
